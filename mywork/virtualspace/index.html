Fossil virtual meeting space

Type

Personal, class project

Tools

Blender, Mozilla Hub

Timeline

2 weeks; Mar 13 - 27, 2023


This low poly fossil site model was created for a virtual environment built in Spoke by Mozilla Hubs. Multiple users can join and explore the environment or use it as a virtual meeting room.

The design process includes modeling the assets in Blender, baking texture, and then importing them into Spoke.
